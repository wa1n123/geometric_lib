# Geometric_lib
## <h2 style="color: #007bff; margin-top: 20px;">Общее описание</h2>
Библиотека предоставляет удобные функции для вычисления площади и периметра базовых геометрических фигур. Она позволяет упростить математические расчеты при работе с геометрическими объектами.
## <h2 style="color: #007bff; margin-top: 20px;">Доступные фигуры</h2>
- Круг
- Квадрат  
- Прямоугольник
- Треугольник
## <h2 style="color: #007bff; margin-top: 20px;">Основные возможности</h2>
- Вычисление площади фигур
- Вычисление периметра фигур
- Подробная документация для каждой функции
## <h2 style="color: #007bff; margin-top: 20px;">Доступные функции и примеры их работы</h2>
### <h2 style="color: #007bff; margin-top: 20px;">Работа с кругом</h2>
```python
import math


def area(r):
    """
    Вычисляет площадь круга.
    
    Parameters:
        r (float): Радиус круга
        
    Returns:
        float: Площадь круга по формуле π * r^2
    """
    return math.pi * r * r


def perimeter(r):
    """
    Вычисляет длину окружности (периметр круга).
    
    Parameters:
        r (float): Радиус круга
        
    Returns:
        float: Длина окружности по формуле 2 * π * r
    """
    return 2 * math.pi * r
    print(area(4)) # 50.24
    print(perimetr(5)) # 31.4
```
### <h2 style="color: #007bff; margin-top: 20px;">Работа с прямоугольнком</h2>
```python
def perimeter(a, b):
    """
    Вычисляет периметр прямоугольника.
    
    Parameters:
        a (float): Длина первой стороны прямоугольника
        b (float): Длина второй стороны прямоугольника
        
    Returns:
        float: Периметр прямоугольника по формуле 2 * (a + b)
    """
    return (a + b) * 2
    print(area(2,3)) # 6
    print(perimetr(3,4)) # 14
```
### <h2 style="color: #007bff; margin-top: 20px;">Работа с квадратом</h2> 
```python
def area(a):
    """
    Вычисляет площадь квадрата.
    
    Parameters:
        a (float): Длина стороны квадрата
        
    Returns:
        float: Площадь квадрата по формуле a * a
    """
    return a * a


def perimeter(a):
    """
    Вычисляет периметр квадрата.
    
    Parameters:
        a (float): Длина стороны квадрата
        
    Returns:
        float: Периметр квадрата по формуле 4 * a
    """
    return 4 * a
    print(area(4)) # 16
    print(perimetr(3)) # 12
```
### <h2 style="color: #007bff; margin-top: 20px;">Работа с треугольником</h2>
```python
def area(a, h):
    """
    функция по поиску площади треугольника
    входные данные: высота и основание треугольника
    возвращает его площадь
    """
    return a * h / 2

def perimeter(a, b, c):
    """
    функция по поиску площади треугольника
    входные данные: стороны треугольника
    возвращает его периметр
    """
    return a + b + c
    print(area(3,4)) # 6
    print(perimetr(3,4,5)) # 12
```
# <h2 style="color: #007bff; margin-top: 20px;">Математика</h2>

## Площадь

| Фигура       | Формула              |
|--------------|----------------------|
| Круг         | S = π × r²          |
| Квадрат      | S = a²              |
| Треугольник  | S = (a × h) / 2     |
| Прямоугольник| S = a × b           |

## Периметр

| Фигура       | Формула              |
|--------------|----------------------|
| Круг         | P = 2 × π × r       |
| Квадрат      | P = 4 × a           |
| Треугольник  | P = a + b + c       |
| Прямоугольник| P = 2 × (a + b)     |
